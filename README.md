# Nexiora Certificate Generator System

A modern full-stack certificate generation platform built using HTML, CSS, JavaScript, Node.js, and Express.js.

The system allows admins to generate professional certificates dynamically, download them as PDFs, and manage records through an admin dashboard.

---

## Features

- Admin Signup & Login Authentication
- Protected Dashboard Access
- Dynamic Certificate Generation
- PDF Download Support
- Professional Certificate UI
- Admin Dashboard
- Certificate Record Management
- Search & Delete Functionality
- Responsive Modern UI
- Digital Signature Integration
- Unique Certificate IDs

## Installation

Clone the repository:

Navigate to project folder:

Install dependencies:

npm install

Start the server:

node server.js
